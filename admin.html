🔐 Acesso restrito

Este painel funciona com o seu token pessoal do GitHub — só quem tem o token consegue publicar mudanças no app.

Como criar o token (uma vez só):
1. No GitHub (logado como My-Ticket-tech), clique na sua foto no canto superior direito → Settings
2. Desça o menu da esquerda até o fim: </> Developer settings
3. Personal access tokens → Fine-grained tokens → Generate new token
4. Preencha: Token name: painel-glow · Expiration: Custom → 1 ano
5. Em Repository access, marque Only select repositories e escolha glow-app
6. Clique em Repository permissions (lista azul) e procure a linha Contents → mude de "No access" para Read and write (as outras linhas ficam como estão)
7. Botão verde Generate token no final → copie o código que aparece (começa com github_pat_) e cole aqui embaixo.
Atalho direto: github.com/settings/personal-access-tokens/new

O token fica salvo apenas neste navegador, no seu aparelho.
